## ⚙️ Installation & Setup  

### Clone the repo  
```bash
git clone https://github.com/GitSetGoRishabh/sih-2025-trackmitra.git
cd sih-2025-trackmitra
```

### Backend setup
```bash
cd backend
npm install
npm start
```

### Frontend setup
```bash
cd frontend
npm install
npm run dev
```
